Lo's Docs är ett skolprojekt (2025) där jag återskapade Tailwind CSS CLI-dokumentationssidan med fokus på att **följa en given design så exakt som möjligt**, bygga responsivt och jobba utility-first med Tailwind.

Projektet är mest statiskt, men innehåller några interaktiva delar:
- Dark/Light/System theme toggle
- Search-modal
- Responsiv meny + sidebar (mobil)
- Hover/focus-states för UI-detaljer

## Fokus i projektet
- **Given design → implementation** (pixel/perception accuracy)
- Tailwind CLI-flöde (input → output CSS)
- Responsiv layout (desktop + mobile)
- UI-states (hover/focus/active, dark mode)
- Ren struktur och tydliga komponentdelar i markup

## Värt att veta innan du klickar runt
Det här är en **statisk demo**, vilket betyder:
- inga backend-funktioner
- inga riktiga sökresultat (search-modal är UI/interaction)
- länkar/sektioner är främst för layout och känsla (doc-site look & feel)
